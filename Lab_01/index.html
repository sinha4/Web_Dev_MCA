1991: The Birth of the Web

The web was born when Tim Berners-Lee created the first website using basic <html>, <head>, and <body> tags.

Quick Fact

“The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect.”
— Tim Berners-Lee

1995: JavaScript Joins the Party

Netscape introduced JavaScript, enabling early interactivity. It ran on the client side and revolutionized user experience.

Fun Fact

JavaScript was created in just 10 days by Brendan Eich!

1996: CSS Appears

CSS allowed developers to separate content from design, introducing new levels of customization to the static web.

Originally proposed by Håkon Wium Lie
while at CERN, Switzerland
